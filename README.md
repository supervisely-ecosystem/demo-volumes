Demo project with dicom / nrrd volumes without labels
## Overview 

`demo_volumes` is an example project with 3 unlabeled volumes. 
Volumes for this project were acquired via [3D Slicer](https://www.slicer.org/) application.


Included volumes:
* CTACardio
* CTChest
* MRHead
